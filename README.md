# Learning Projects

Repositório destinado a pequenos projetos, estudos, experimentos e testes desenvolvidos durante minha jornada como desenvolvedor.

Aqui você encontrará aplicações simples, exercícios práticos e projetos criados para explorar novas tecnologias, conceitos e ideias.

## Objetivo

Este repositório funciona como um laboratório de aprendizado, onde posso:

- Praticar programação
- Testar novas tecnologias
- Explorar conceitos e padrões de desenvolvimento
- Registrar minha evolução como desenvolvedor

## Estrutura

Os projetos são organizados em pastas independentes, cada uma contendo seu próprio código-fonte e documentação quando necessário.
